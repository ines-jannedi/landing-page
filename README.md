# Modern UX/UI Landing Page

A modern and responsive landing page built with React.js and Tailwind CSS.  
This project focuses on clean UI design, smooth layout structure, and responsive user experience across all devices.

---

## 🚀 Live Demo
[View Project](https://landing-page-beta-rose-lfytinaaeo.vercel.app)

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Vite

---

## ✨ Features
- Fully responsive design (mobile, tablet, desktop)
- Modern UI/UX layout
- Reusable React components
- Clean and minimal design
- Smooth navigation and section transitions
- Fast performance and optimized structure
